# Crossword-Puzzle
#### This is a crossword puzzle game written in python. When you run the program there are three options:(G) Guess Word, (C) Show Clue, (R) Show Grid, (Q) Quit. Guess word allows you to guess a word on the grid. Show clue allows you to receive a hint for any word on the grid. Show Grid allows the player to view the grid, and Quit will exit the game. There are 84 words to guess and they are imported through a .csv file and then displayed onto the crossword grid. The grid is composed of ASCII characters and each character in a word that has not yet been guessed is substituted with an asterisk.
